## [Amazon Elastic Container Registry (ECR)](https://docs.aws.amazon.com/AmazonECR/latest/userguide/what-is-ecr.html)
Amazon ECR is an AWS managed container image registry service that supports private and public images.
Amazon ECR supports private repositories with resource-based permissions using AWS IAM. It supports vulnerability scanning, versioning, image tags,
image lifecycle, encryption, and more.
