## [Amazon Elastic Kubernetes Service (EKS)](https://docs.aws.amazon.com/eks/latest/userguide/what-is-eks.html)
Amazon Elastic Kubernetes Service (EKS) provides a fully managed Kubernetes service that eliminates the complexity of operating Kubernetes clusters.

### Node Types
- **Managed Node Groups:** EC2 instances managed by AWS, Nodes are part of Auto Scaling Group, Support for On-Demand and Spot Instances
- **Self-Managed:** Own managed instances registered in a clustered and managed by an Auto Scaling Group
- **Fargate:** Serverless compute
