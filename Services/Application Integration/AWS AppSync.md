## [AWS AppSync](https://docs.aws.amazon.com/appsync/latest/devguide/what-is-appsync.html)

AWS AppSync is a managed GraphQL API service that securely queries and aggregate data from multiple data sources
such as DynamoDB, Lambda, RDS, and HTTP APIs.

### [AppSync Resolvers](https://docs.aws.amazon.com/appsync/latest/devguide/resolver-components.html)

Resolvers are functions that map a specific GraphQL field to its underlying data source, defining how to fetch, write,
or transform data for that field.

### [AppSync Authentication](https://docs.aws.amazon.com/appsync/latest/devguide/security-authz.html)

AppSync supports multiple authentication methods including API keys, IAM, OpenID Connect, and Cognito User Pools.
