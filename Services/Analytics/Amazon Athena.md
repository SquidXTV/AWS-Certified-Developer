## [Amazon Athena](https://docs.aws.amazon.com/athena/latest/ug/what-is.html)

Amazon Athena is an interactive query service that makes it easy to analyze data directly in
Amazon S3 using standard SQL.

### [Athena Federated Queries](https://docs.aws.amazon.com/athena/latest/ug/federated-queries.html)

Federated Queries let you run SQL queries in Athena against data outside of S3, such as relational databases,
NoSQL stores, or other sources, by using data source connectors that pull data on demand. Athena then joins
and analyzes data across these sources as if they were a single dataset.
