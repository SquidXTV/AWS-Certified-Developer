## [AWS AppConfig](https://docs.aws.amazon.com/appconfig/latest/userguide/what-is-appconfig.html)

AWS AppConfig helps you safely change application behavior in production without redeploying code.
Using feature flags and dynamic free-form configurations, you can control how your application runs in real time.

### [AppConfig Validators](https://docs.aws.amazon.com/appconfig/latest/userguide/appconfig-creating-configuration-and-profile-validators.html)

A validator ensures that your configuration data is syntactically and semantically correct using either AWS Lambda Functions or JSON Schema.
